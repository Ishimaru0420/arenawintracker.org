02Welche Daten werden verarbeitet

Für den Betrieb werden folgende Daten verarbeitet:

Riot-ID (Name#Tag)Um deinen Spieler-Account bei der Riot-API zu identifizieren und deinen Sync zuzuordnen.
RegionServer-Region deines Accounts (z.B. Europe), zur richtigen Riot-API-Abfrage.
Match-Historie (Arena)Datum, Champion, Platzierung, genutzte Items/Augments/Perks/Summoner-Spells und Teammates deiner Arena-Spiele seit Season-Start.
Season-Start-DatumEinstellung, die du selbst festlegst.
Profilicon-IDDein öffentliches Riot-Profilbild - identisch zu dem, was ohnehin im Spiel und über die Riot-API für jeden sichtbar ist.
FreundeslisteDie Riot-IDs der von dir selbst hinzugefügten Freunde, damit dir ein Freundes-Ranking angezeigt werden kann.
IP-AdresseWird kurzfristig technisch verarbeitet, um Registrierung und manuellen Sync vor Missbrauch zu schützen (Rate-Limit). Sie wird nicht dauerhaft gespeichert oder mit deiner Riot-ID verknüpft abgelegt.

Du musst keinen eigenen Riot API-Key angeben - der Abruf bei Riot erfolgt zentral über den Sync-Server. Es werden keine Zahlungsdaten, keine Standortdaten und keine Klartextdaten von anderen Spielern außerhalb deiner eigenen Match-Historie (z.B. Teammate-Namen aus euren gemeinsamen Matches) verarbeitet.

03Wo deine Daten gespeichert werden

Arena Win Tracker nutzt einen zentralen Sync-Server statt lokaler Speicherung. Das betrifft sowohl die Overwolf-App als auch die Web-App-Version.

Zentrale Datenbank Deine Riot-ID, Region, Season-Start, Profilicon, Freundesliste sowie deine Match-Historie und Sieg-Übersicht werden in einer MongoDB- Atlas-Datenbank gespeichert (Rechenzentrum: AWS Frankfurt, EU). Der Sync-Server selbst läuft bei Render, ebenfalls in der Region Frankfurt (EU). Deine Daten verlassen damit für den Normalbetrieb die EU nicht.
Lokale Zwischenspeicherung Zusätzlich speichert dein Browser eine Kopie der zuletzt geladenen Anzeige-Daten lokal (localStorage), damit die Oberfläche auch ohne erneuten Serverabruf sofort etwas anzeigen kann. Das ist nur ein Anzeige-Cache, keine eigenständige Datenquelle, und lässt sich über die Einstellung "Lokale Anzeige zurücksetzen" jederzeit löschen.

04Wer deine Daten sehen kann

Arena Win Tracker ist als kleine, öffentlich erreichbare Webseite mit zugehörigem Server aufgebaut. Das bedeutet konkret:

Globales Ranking Das Globale Ranking auf der Webseite ist für jeden Besucher öffentlich sichtbar, auch ohne eigene Registrierung - inklusive deiner Riot-ID, deines Profilicons und deiner Sieg-Statistiken (Anzahl Champions, Gesamt-Siege, bester Champion). Wenn du das nicht möchtest, kontaktiere uns für eine Löschung (siehe Abschnitt 06).
Detail-Statistiken Deine vollständige Match-Historie ist über deine Riot-ID technisch abrufbar, ähnlich wie öffentliche Drittanbieter-Seiten zu League of Legends (z.B. op.gg) ohnehin auf Basis öffentlicher Riot-API-Daten funktionieren. Es gibt keinen Login-Schutz, der das einschränkt.
Freunde Nutzer, die dich als Freund hinzugefügt haben, sehen deinen Fortschritt zusätzlich im Freundes-Ranking - das ist der eigentliche Zweck der Freundesfunktion.

05Riot Games API & weitere Drittanbieter

Zur Abfrage der Match-Daten nutzt der Server die offizielle Riot Match-V5 API sowie Champion-Daten von Riots Data Dragon CDN. Für den Betrieb und einzelne Funktionen werden zusätzlich folgende Dienste eingesetzt:

Render (Hosting, EU)Betreibt den Sync-Server und den automatischen Abgleich.
MongoDB Atlas (Datenbank, EU)Speichert die in Abschnitt 02 genannten Daten dauerhaft.
blitz.ggQuelle für die angezeigten externen Meta-Trios. Es werden keine deiner Daten an blitz.gg übermittelt, die Daten fließen ausschließlich in eine Richtung (Import durch uns).
OpenRouter (KI-Dienst)Wird für die experimentelle KI-Analyse-Funktion genutzt. Dabei werden ausschließlich aggregierte, anonyme Champion-Statistiken zur Auswertung übermittelt - keine Riot-IDs, keine individuellen Match-Daten und keine sonstigen personenbezogenen Angaben.
Ko-fi (freiwillige Spenden)Der "☕ Unterstützen"-Link führt auf eine externe Ko-fi-Seite. Es werden dabei keine Daten von uns an Ko-fi übermittelt - sobald du dem Link folgst, gilt die Datenschutzerklärung von Ko-fi für deine dortigen Angaben.

Es werden keine Werbenetzwerke und keine Analyse- oder Tracking-Dienste zur Nutzerverfolgung eingebunden.

06Deine Rechte & Löschung

Die App-Einstellung "Lokale Anzeige zurücksetzen" löscht nur den lokalen Anzeige-Cache auf deinem Gerät - nicht die zentral auf dem Server gespeicherten Daten. Für Auskunft, Berichtigung oder vollständige Löschung deiner auf dem Server gespeicherten Daten (inklusive deines Eintrags im globalen Ranking) wende dich jederzeit an die oben genannte Kontaktadresse; die Löschung erfolgt dann zeitnah manuell durch den Verantwortlichen.

07Speicherdauer

Deine Daten werden gespeichert, solange dein Account besteht. Es gibt aktuell keine automatische Löschung nach einer festen Frist oder bei Inaktivität - eine Löschung erfolgt auf Anfrage gemäß Abschnitt 06.
