02Welche Daten werden verarbeitet

Für den Betrieb der App werden folgende Daten verarbeitet:

Riot-ID (Name#Tag)Um deinen Spieler-Account bei der Riot-API zu identifizieren.
Riot API-KeyDein eigener, persönlicher Schlüssel zur Authentifizierung gegenüber der Riot-API.
Match-Historie (Arena)Datum, Champion, Platzierung und Teammates deiner Arena-Spiele seit Season-Start.
Region & Season-Start-DatumEinstellungen, die du selbst festlegst.

Es werden keine Zahlungsdaten, keine Standortdaten und keine Daten von anderen Spielern außerhalb deiner eigenen Match-Historie verarbeitet.

03Wo deine Daten gespeichert werden

In der aktuellen Version läuft Arena Win Tracker als lokale Overwolf-App. Alle oben genannten Daten werden ausschließlich im lokalen Speicher (localStorage) deines eigenen Rechners abgelegt. Es findet keine Übertragung an einen Server von uns statt.

Sollte zu einem späteren Zeitpunkt eine Server-Komponente für mehrere Nutzer angeboten werden, wird diese Erklärung aktualisiert und beschreibt dann zusätzlich, welche Daten auf dem Server verarbeitet werden und wie lange sie gespeichert bleiben.

04Riot Games API & Drittanbieter

Zur Abfrage deiner Match-Daten nutzt die App die offizielle Riot Match-V5 API sowie Champion-Daten von Riots Data Dragon CDN. Dabei gelten zusätzlich die Datenschutzbestimmungen von Riot Games. Es werden keine eigenen Analyse- oder Tracking-Dienste, Werbenetzwerke oder sonstigen Drittanbieter eingebunden.

05Deine Rechte

Da alle Daten lokal bei dir gespeichert sind, hast du jederzeit volle Kontrolle: über die Einstellungen der App kannst du alle gespeicherten Daten per Klick auf "Daten zurücksetzen" vollständig löschen. Für Fragen zu deinen Rechten (Auskunft, Löschung, Widerspruch) kannst du dich jederzeit über die oben genannte Kontaktadresse melden.

06Änderungen dieser Erklärung

Diese Datenschutzerklärung wird angepasst, sobald sich an der Datenverarbeitung etwas ändert, zum Beispiel durch Einführung der geplanten Server-Komponente. Es lohnt sich, bei größeren Updates der App hier vorbeizuschauen.
