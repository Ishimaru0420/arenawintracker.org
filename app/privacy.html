02Welche Daten werden verarbeitet

Für den Betrieb werden ausschließlich folgende Daten verarbeitet:

Riot-ID (Name#Tag) Zur Identifikation deines Accounts bei der Riot-API und zur Zuordnung deines täglichen Syncs.
Region Deine Server-Region (z. B. Europe), die für die korrekte Riot-API-Abfrage benötigt wird.
Match-Historie (Arena) Datum, Champion und Platzierung deiner Arena-Spiele seit dem von dir gewählten Season-Start-Datum.
Season-Start-Datum Eine Einstellung, die du selbst in der App festlegst und jederzeit ändern kannst.
Profilbild-ID Die ID deines In-Game-Profilbilds, ausschließlich zur Anzeige im Leaderboard der App.

Du musst keinen eigenen Riot API-Key angeben – der Abruf bei Riot erfolgt zentral über den Sync-Server. Es werden keine Zahlungsdaten, keine Standortdaten und keine Passwörter gespeichert.

03Wo deine Daten gespeichert werden

Zentrale Datenbank Riot-ID, Region, Season-Start sowie Match-Historie und Sieg-Übersicht werden auf einem Server gespeichert (Hosting: Render, Frankfurt; Datenbank: MongoDB Atlas, Frankfurt), damit der tägliche automatische Sync funktioniert.
Lokaler Cache Dein Browser speichert eine Kopie der zuletzt geladenen Anzeige-Daten in localStorage. Das ist nur ein Anzeige-Cache, keine eigenständige Datenbank – er kann in den App-Einstellungen jederzeit geleert werden.

Andere registrierte Nutzer können im Leaderboard deine Riot-ID und Sieg-Statistiken sehen – genauso wie du die der anderen Spieler siehst. Deine vollständige Match-Historie ist nur abrufbar, wenn jemand gezielt deine Riot-ID eingibt.

04Riot Games API & Drittanbieter

Zur Abfrage der Match-Daten nutzt der Server die offizielle Riot Match-V5 API sowie Champion- und Patch-Daten von Riots Data Dragon CDN. Folgende Dienste werden für den Betrieb eingesetzt:

RenderBetreibt den Sync-Server (Frankfurt). Datenschutzerklärung: render.com/privacy
MongoDB AtlasSpeichert die in Abschnitt 02 genannten Daten (Frankfurt). Datenschutzerklärung: mongodb.com/legal/privacy-policy
OverwolfPlattform für die Overlay-Version der App. Eigene Datenschutzerklärung: overwolf.com/legal/overwolf-apps-privacy-policy

Es werden keine Werbenetzwerke, Analyse- oder Tracking-Dienste von Drittanbietern eingebunden. In der Overwolf-Version können Overwolf-eigene Anzeigen erscheinen, die ausschließlich Overwolfs eigenen Richtlinien unterliegen.

05Deine Rechte (DSGVO)

Du hast das Recht auf Auskunft, Berichtigung und vollständige Löschung deiner auf dem Server gespeicherten Daten. Die Einstellung „Lokale Anzeige zurücksetzen" in der App löscht nur den lokalen Cache auf deinem Gerät – nicht die serverseitig gespeicherten Daten.

Für die Löschung deiner Serverdaten wende dich an die oben genannte Kontaktadresse. Die Löschung erfolgt innerhalb von 14 Tagen.
